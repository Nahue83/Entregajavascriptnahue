README – Entrega JavaScript

Este proyecto es un simulador de ecommerce desarrollado para la segunda entrega del curso de JavaScript.
Permite mostrar productos de manera dinámica, agregarlos al carrito y gestionar la compra utilizando DOM, eventos y localStorage.

Funcionalidades

Renderizado de productos desde arrays de objetos.

Agregar productos al carrito.

Aumentar, disminuir o eliminar cantidades.

Vaciar el carrito.

Finalizar compra.

Guardado del carrito en localStorage para mantener los datos al refrescar la página.

Cómo usarlo

Entrar a cualquiera de las categorías.

Elegir un producto y presionar Agregar.

Abrir el carrito para ver los productos seleccionados.

Manejar cantidades o vaciar el carrito.

Finalizar compra para ver el total.
